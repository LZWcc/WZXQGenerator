# 旺仔小乔名字生成器

---

一个趣味十足的前端小项目，随机生成由“饮料名称”+“王者荣耀英雄名字”组成的爆款网络名字，如：

- 可乐司空震
- O泡西施
- 旺仔小乔

点击按钮即可生成，也支持根据名字长度筛选，还支持复制和历史记录保存！

(主要目的为练习JavaScript, css由copilot编写)

👉 在线体验地址：[点击这里访问](https://lzwcc.github.io/WZXQGenerator/)
